Basic HTML Tags

Tag Example Description
<html> <html></html> Represents the root element of an HTML document.
<head> <head></head> Contains meta-information about the HTML document.
<title> <title>Learn HTML</title> Sets the title of the HTML document shown in the browser tab.
<meta> <meta name="author" content="John Doe"> Provides metadata about the HTML document.
<link> <link rel="stylesheet" href="styles.css"> Defines a link between the HTML document and an external resource.
<body> <body>Content goes here</body> Contains the content of the HTML document that is displayed in the browser.
<h1> to <h6> <h1>Heading</h1> Defines headings of various levels.
<p> <p>This is a paragraph.</p> Represents a paragraph of text.
